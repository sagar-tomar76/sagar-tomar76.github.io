Pop Drops

Enjoy 100% TNT drop rate, a Java exclusive feature, in Minecraft PE/BE using the Pop Drops addon. This addon is very easy to install and has many uses for those who like making farms because it will allow you to collect all of the blocks that TNT blows up.

Features:
> Download the addon.
> Import it into Minecraft.
> Add it to your world.
> Make sure "Beta APIs" experimental toggle is enabled like in the picture below:

> Enjoy!
